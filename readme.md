# git操作

## 初始化版本库
- git init
- 初始化成功后，当前目录后面有(master)
- 初始化成功后当前目录下有个隐藏文件.git(不要管，不要操作这个文件)

## 工作区
- 持有实际文件
- 我们平时增删改查的文件都是工作区的内容

## 暂存区
- 可以理解为我们看到的一个地方
- 也是存在于用户电脑中的
- 本地仓库的一个主要组成部分

## 本地仓库
- 可以理解为我们看到的一个地方
- 也是存在于用户电脑中的
- 用于存储项目代码及版本项目记录等信息

## 提交到暂存区
- git add 文件名
- 将工作区的变动提交到暂存区
- git add .  将所有变动提交到暂存区

## 查看状态
- git status
- 查看工作区和暂存区的状态(增删改)

## 提交到本地仓库
- git commit -m '提交注释'
- 将代码从暂存区提交到本地仓库
